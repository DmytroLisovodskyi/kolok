# Webinar Platform

## Description
Platform for conducting online webinars and connecting users to virtual rooms.

---

## Technologies
- Docker
- Nginx
- GitHub Actions

---

## Installation

### Build container

```bash
docker build -t webinar-platform .
```

### Run container

```bash
docker run -p 8080:80 webinar-platform
```

---

## Project Structure

```text
README.md
Dockerfile
index.html
.github/workflows/main.yml
```

---

## Developer Quick Start

1. Clone repository
2. Build Docker image
3. Run container
4. Open browser on localhost:8080
